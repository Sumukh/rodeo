# Rodeo - An Automatic Data Wrangler

## Setup

Python 3

```
pip install virtualenv

virtualenv -p python3 env
source env/bin/activate

pip install -r requirements.txt
```

## Usage

```
source env/bin/activate

cd src
python3 rodeo.py
```

## Limitations

Consider this a proof of concept. Extracts data from CSVs.
